## ✨ Features
- 🤖 AI Credit Scoring (XGBoost + Random Forest)
- 🔍 Explainable AI — SHAP-based XAI decision explanation
- 🚨 Anomaly Detection (Isolation Forest)
- 👤 Human-in-the-Loop Review for borderline cases
- ✅ Rule-Based Validation (8+ RBI compliance rules)
- 📧 Real-time Gmail Notifications
- 🧮 EMI Calculator with Amortization Schedule
- 📊 Admin Dashboard with verification panel
- 💬 AI Chatbot (24×7 loan query assistant)

## 🛠 Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | HTML5, CSS3, Vanilla JavaScript |
| AI Model | XGBoost, Random Forest, SHAP |
| Anomaly Detection | Isolation Forest |
| Notifications | Gmail API / SMTP |
| Database | PostgreSQL |
| Deployment | GitHub Pages |
