# 🏦 SBI MSME AI Credit Risk & Loan Portal

An AI-powered loan approval system for MSMEs built for SBI.

## Features
- 🤖 AI Credit Scoring (XGBoost)
- 🔍 Explainable AI (SHAP/XAI)
- 🚨 Anomaly Detection (Isolation Forest)
- 👤 Human-in-the-Loop Review
- 📧 Gmail Notifications
- 🧮 EMI Calculator
- 📊 Admin Dashboard
